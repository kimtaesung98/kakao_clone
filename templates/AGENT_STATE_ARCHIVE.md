## 아카이브 규칙

- `AGENT_STATE.md` 누적 요구사항 **20행 초과 시** → 가장 오래된 항목부터 이 파일로 이동
- 이동 후 `AGENT_STATE.md`의 해당 행 삭제
- 이 파일 자체는 삭제하지 않고 계속 누적
- 이 파일이 100행을 초과하면 날짜 기준으로 별도 연도별 파일로 분리 가능  
  예: `AGENT_STATE_ARCHIVE_2026.md`
